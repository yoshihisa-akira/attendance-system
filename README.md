# attendance-system
##　概要説明
勤怠管理システムです。
出勤退勤時間や休憩開始終了時間を記録します。

##　作成した目的
実践学習タームを通して実際の案件について知るため。

##　アプリケーションURL
localhostでログイン画面に飛びます。

##　機能一覧
ログイン機能、登録機能、出勤退勤時間打刻機能、休憩開始終了打刻機能。

##　使用技術
laravel 8.83.27, php 7.4.9
